# Supplementary Documents for Exercism Scientific Tracks

[Exercism][exercism] is an educational non-profit aiming to help students improve their programming fluency across a wide range of languages (currently 80+, and still growing).

Within the community of Exercism maintainers, those of us who work on the [Julia][ex-julia], [Python][ex-python] and [R][ex-r] (JuPyteR) tracks have a particular interest in scientific computing and data science.
Some of this interest is expressed in the teaching syllabus documents on the Exercism website.
At the more technical end, we run up against the limits of what Exercism is designed to do.

This GitHub Pages site is an experiment to let us explore the wider capabilities of these three languages: both for our own self-education, and to to provide a "See Also" resource for the minority of student who want to learn more.
